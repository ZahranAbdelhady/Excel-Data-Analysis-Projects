# Excel-Data-Analysis-Projects
A collection of data cleaning, auditing, and analysis tasks using Microsoft Excel.
# Excel Data Analysis & Cleaning Projects

المستودع يحتوي على مجموعة من المهام العملية لتنظيف وتحليل البيانات باستخدام برنامج Microsoft Excel. تم تنفيذ هذه المهام لتطبيق مهارات معالجة البيانات، واستخدام المعادلات المنطقية، والتنسيق الشرطي.

## Task 1: Basic Data Entry & Formula Calculations (Task 1.xlsx)
* إنشاء ورقة عمل تحتوي على بيانات تسلسلية وعشوائية لتواريخ وأوقات تسليم وكميات.
* حساب المبالغ المستحقة باستخدام المعادلات الرياضية بناءً على أسعار المواد.
* حساب المبالغ المدفوعة بنسبة 35% من المبلغ المستحق، وحساب المبلغ المتبقي.
* إنشاء جدول ملخص إحصائي يوضح عدد الأنواع، مجموع الكميات، مجموع المبالغ المستحقة، معدل المبالغ المستحقة، ومجموع المبالغ المتبقية.

## Task 2: Data Cleaning & Formatting (Task 2.xlsx)
* تنظيف بيانات عقود الموظفين وتصحيح الأخطاء الإملائية في أسماء أماكن العمل.
* تنسيق التواريخ لتظهر بصيغة مخصصة (رقم اليوم، اسم الشهر، السنة)، وتنسيق عمود الأجور بالريال السعودي مع رقم عشري واحد.
* توزيع البيانات على 4 أوراق عمل مختلفة (يومي، نصف شهري، شهري، نصف سنوي) باستخدام أداة التصفية (Filter).
* تحويل نطاق البيانات إلى جدول منظم وتفعيل صف الإجمالي (Total Row) لحساب عدد الموظفين، وأول وآخر تاريخ توقيع، ومعدل الأجور].
* حساب النتائج الإحصائية باستخدام الطريقة الاعتيادية وباستخدام اسم الجدول في المعادلات.

## Task 3: Logical Functions & Conditional Formatting (Task 3.xlsx)
* استخدام المعادلات المنطقية لإعطاء درجات وتقييم استحقاق العوائل للمعونة بناءً على شروط متعددة مثل: عدد الأبناء، وجود إعاقة، وحالة العمل.
* تمييز البيانات الهامة باستخدام التنسيقات الشرطية (Conditional Formatting) مثل تمييز الأمهات المعيلات وأعلى 10 قيم لعدد الأبناء].
* حساب مبالغ المعونة المستحقة وربطها بخلية مرجعية (Absolute Reference) لسهولة التحديث.
* استخدام الأشرطة اللونية (Data Bars) لتمثيل درجات وضع العائلات بصرياً.
* استخدام ميزة الفلترة لإجراء عمليات العد الإحصائي للعوائل التي بها إعاقة أو العوائل غير العاملة، وترتيب البيانات بناءً على درجات الاستحقاق.
